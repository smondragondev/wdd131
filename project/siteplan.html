Site Name

Ferreñafe Professional Services Center

This name represents that this page will be a center to provide professional services in the Ferreñafe area.

Domain availability: feproser.com

Site Purpose

The purpose of this site is to connect users with professional services in the Ferreñafe area, increasing the visibility of local professionals who offer services such as driving, tutoring, painting, and more.

Scenarios

  • Where can a user find a local professional?
  • Where can a local professional join the portal?
  • Where can a user contact a local professional?

Color Schema

  • Primary Color: Yellow Green (#A7C957)
  • Secondary Color: Dark Cerulean (#05458A)
  • Accent Color: Drab dark brown (#363C16)
  • Accent Color 2: Snow Flurry (#DFF8D5)
  • Typeface Color: Rich Black (#011627)
  • Background Color: Ghost White (#F0EFF4)

For headings, the primary and accent color will be used. For body text, the typeface color will be used. The background color will be applied to the main content areas. For call to action elements, the secondary color will be used.

Typography

The site will use the Noto Sans font for body text and Noto Serif for headings. The font-weight will vary depending on hierarchy.
